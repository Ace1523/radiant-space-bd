# Radiant Space BD Hub

A comprehensive business development resource for the space nuclear market, built by Radiant Nuclear.

**Live Site:** https://radiantspace.netlify.app

## What Is This?

An internal BD tool consolidating everything needed to navigate space nuclear contract opportunities:

- **Contract Opportunities** - Active RFIs, BAAs, solicitations with deadlines
- **Agency Guides** - NASA, DARPA, Space Force, DIU, DOE, Air Force, NRO
- **Strategy & Playbook** - Capture planning, teaming strategy, proposal guides
- **Technical Overview** - Fission 101, reactor design, HALEU, power conversion
- **Programs & Timeline** - FSP, DOME, Pele, JETSON, and more
- **Policy & Regulation** - Launch approval, export controls, safety frameworks
- **News & Events** - Industry updates, conference calendar
- **160+ Term Glossary** - Searchable reference for acronyms and concepts

## Tech Stack

Single-page static site:
- HTML/CSS/JavaScript (all in one file)
- No backend required
- Hosted on Netlify + GitHub Pages

## Deployment

Both auto-deploy on push to `main`:
- **Netlify:** https://radiantspace.netlify.app
- **GitHub Pages:** https://ace1523.github.io/radiant-space-bd/

## Local Development

Just open `index.html` in a browser - it's fully self-contained.
